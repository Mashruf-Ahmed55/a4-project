Question-1:What are the different ways to select an element in the DOM?
There are several methods available for selecting elements in the DOM:document.getElementById("id") - Selects an element using its unique ID,document.getElementsByClassName("class") - Selects all elements that share the same class name,document.querySelector("selector") - Selects the first element that matches a given CSS selector,document.querySelectorAll("selector") - Selects all elements that match a given CSS selector.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
HTML elements have three main properties for handling text and content innerText retrieves only the visible text within an element, ignoring hidden content (display: none)innerHTML allows reading or modifying the full HTML content, including all child elements and tags.textContent retrieves all text inside an element, including hidden text, but treats it as plain text without rendering HTML tags.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a single event listener is added to a parent element, allowing it to handle events from its child elements. Instead of adding multiple event listeners to individual child elements, event delegation improves performance by reducing memory usage and simplifying event handling.
Question-3: What is event bubbling in the DOM?
Event bubbling is the process where an event triggered on an element propagates upward through its parent elements. The event starts from the target element and moves up the DOM tree unless event.stopPropagation() is used to stop it.
